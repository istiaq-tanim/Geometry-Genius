What is DOM and what is the purpose of DOM?

DOM is Document Object Model, which is a programming interface for HTML and XML. DOM represents our homepage as if we can easily modify our homepage by using programing language. We use DOM for user Interactivity. By using DOM we easily create new elements, modify elements and delete elements.

How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML elements by using their class name,id name and tagname.There are five ways we can select HTML elements by using DOM.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

What is a callback function and why will you use it?

Callback function is a function which sent a function to another function as an arguments.It also allows a function to call another function.Callback function can run after another function has executed.
